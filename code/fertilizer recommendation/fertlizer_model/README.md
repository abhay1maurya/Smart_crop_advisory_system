**AI-Powered Precision Nutrient Management for Agriculture**

This module is a high-performance machine learning system designed to recommend the optimal fertilizer for crops based on specific soil chemistry, crop type, and environmental conditions. It uses an **Ensemble Learning Architecture** (Voting Classifier with Random Forest & Gradient Boosting) to achieve superior accuracy and robustness.

Crucially, it includes a **"Farmer-Friendly Logic Layer"** that translates raw chemical predictions (like "Complex NPK") into actionable, specific advice (e.g., "Use Urea because your Nitrogen is low").

ðŸ§  How the Logic Works
The system doesn't just output raw class labels. It applies a Post-Processing Logic Layer:

Prediction: The Ensemble model predicts the category (e.g., "Complex NPK").

Chemical Analysis: If the category is generic, the system checks specific N-P-K levels provided by the user:

If Nitrogen < 30: Override recommendation to UREA (46-0-0).

If Phosphorus < 30: Override recommendation to DAP (18-46-0).

If Potassium < 30: Override recommendation to MOP (0-0-60).

Translation: It maps synthetic dataset labels (like "120-60-60") into real-world advice ("High Yield NPK Mix").
